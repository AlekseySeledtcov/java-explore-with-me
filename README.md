# java-explore-with-me - Исследуй со мной 

Это веб-сервис на базе Spring Boot позволяющее предлагать события и собирать компанию для совместного время припровождения.
___

![Логотип](main/src/main/resources/logo.png)

## Описание
- Цель проекта

Разработать веб‑приложение — платформу для публикации и поиска интересных событий с возможностью собрать компанию для их посещения. Проект выполняется в рамках дипломного задания и призван закрепить и систематизировать профессиональные навыки разработки на Java.

Основная задача — создать два взаимодействующих сервиса:

основной сервис (управление событиями, пользователями, категориями, заявками и модерацией);

сервис статистики (учёт просмотров, аналитика по запросам к API).

- Целевая аудитория

Организаторы событий, публикующие мероприятия.

Пользователи, ищущие интересные мероприятия и компанию для их посещения. 

- Контекст разработки

Проект создаётся как дипломная работа, направленная на комплексное применение знаний и навыковю В ходе разработки решаются следующие задачи:

Многомодульная архитектура - проект организован как Maven - многомодульное приложение;

Микросервисная архитектура - два независимых сервиса взаимодействуют через HTTP‑клиент;

Работа с API и спецификациями. Реализация строго по Swagger‑спецификациям;

Контейнеризация. Запуск сервисов и баз данных в Docker‑контейнерах (настройка Dockerfile и docker-compose.yml);

Самостоятельное проектирование и реализация дополнительной фичи - комментарии.

- Уникальность 

Комплексный подход к разработке проекта - от проектирования до развертывания в контейнерах. 

Использование стека технологий востребованного в индустрии. 

Поэтапное усложнение задач - от простого сервиса к статистики к основному сервису и дополнительной функциональности. 

Отработка навыков DevOps. Настройка Docker‑контейнеров и оркестрация через docker-compose. 

Фокус на чистоту кода.

## Технологии и инструменты

- **Язык:** Java 21.
- **Сборка и качество кода:** Maven, Checkstyle.
- **Фреймворки:** Spring Boot 3.3.x (Web, Data JPA, Validation, Actuator).
- **БД:** PostgreSQL.
- **ORM:** Hibernate 6.5.
- **HTTP‑клиент**: Spring RestTemplate
- **Библиотеки и утилиты:** Lombok, SLF4J, MapStruct, Apache HttpClient 5, Jackson, Jakarta EE.

## Установка и запуск

### Требования
- Java 21+.
- Maven 3.x.
- Git.
- Docker и Docker Compose.

### Инструкция по установке и запуску через Docker Compose

#### 1. Клонирование репозитория

1. Откройте терминал/командную строку.

2. Перейдите в папку, где хотите разместить проект.

3. Выполните команду:
```bash
git clone https://github.com/AlekseySeledtcov/java-explore-with-me.git
```
4. Перейдите в директорию проекта:

```bash
cd java-explore-with-me
```
#### 2. Сборка проекта

Запустите сервисы через Docker Compose:
```bash
docker-compose up --build
```
#### 3. Проверьте работоспособность

```bash
docker-compose ps
```

#### Доступные сервисы и порты
Сервис	        Порт (Host)	        Назначение

Main Service, порт - 8080, описание - Основная бизнес-логика приложения

Stats Server, порт - 9090, описание - Сервис сбора и хранения статистики

Stats DB, порт - 6541, описание - PostgreSQL база данных сервиса статистики

Main DB, порт - 6542, описание - PostgreSQL база данных основного сервиса

## Остановка и удаление

Чтобы остановить приложение и удалить контейнеры, используйте:

```bash
docker-compose down
```
docker-compose down -v - чтобы удалить тома базы данных
